# vue-admin-template

vue版本的后台模板<Br>
技术栈：Vue.js 2、vue-router、webpack2、iView 2、axios

## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
// For the first time, run init to create index.html
npm run init
npm run dev //本地开发
npm run debug //测试环境
npm run buildDebug //测试环境打包使用，这个无热更新功能
```
### Production(Build)
```bush
npm run build
```
